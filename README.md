## Getting Started

First, clone this repo:

```bash
git clone https://github.com/coderboysobuj/shopit.git
```

Then install package with

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

Run development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
